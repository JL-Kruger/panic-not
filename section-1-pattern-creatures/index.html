‚ö†Ô∏è Crisis Scenarios & Response

Lost/Stolen Device

  1. Use Find My (iOS) or Find My Device (Android) from another device
  2. Mark as lost/Enable Lost Mode
  3. If necessary, remote wipe
  4. Contact carrier to suspend service
  5. Change passwords for important accounts from another device

Compromised Account

  1. Change password immediately
  2. Enable 2FA if not already on
  3. Review recent account activity
  4. Check for unauthorized email forwards/filters
  5. Alert contacts if scam messages were sent

Suspected Malware

  1. Note unusual behavior (battery drain, data usage spikes)
  2. Check battery/data usage for anomalies
  3. Uninstall recently added suspicious apps
  4. Run Play Protect (Android) or review App Privacy Report (iOS)
  5. If persistent: Back up important data, factory reset

Daily

  • Don't leave unlocked in public
  • Use biometric lock for speed
Weekly
  • Restart device once (clears temporary exploits)
  • Check for OS/app updates

Monthly

  • Check battery usage for anomalies
  • Update passwords on rotation
Quarterly
  • Full app audit (delete unused)
  • Review location history
  • Update emergency contacts

Remember:

Security is a spectrum, not a binary. Perfect security means not using a phone at all. The goal is reasonable security without destroying usability.

Good security practices protect against common threats, don't make you paranoid, scale with your actual risk level, and become habits rather than conscious efforts.

The best security measure is awareness. Understanding what the risks are helps you make good decisions about what's worth protecting and how much effort to invest.
